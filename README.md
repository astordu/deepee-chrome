# DeepEE Chrome插件

一个Chrome插件，使用DeepSeek的LLM API提供即时解释。

DeepEE的意思： Deep English Explain。

## 功能
- 根据网页中选取的英文单词或短语，提供关于这个单词或短语的在上下文中的深度解释

## 安装
1. 克隆此仓库
2. 打开Chrome并导航到 `chrome://extensions/`
3. 启用“开发者模式”
<img width="1918" alt="image" src="https://github.com/user-attachments/assets/f795fc47-9d95-45e4-81bb-6a5142da0ed7">

5. 点击“加载已解压的扩展程序”并选择扩展目录

<img width="1740" alt="image" src="https://github.com/user-attachments/assets/aee2809d-e6a3-4989-8b65-797d1f72373b">

7. 点击扩展图标并配置您的DeepSeek API密钥
<img width="1842" alt="image" src="https://github.com/user-attachments/assets/bccdae8d-ba32-4dcd-a6be-e26e830e54bb">


## 使用方法
1. 在网页上选择任意文本
2. 点击出现的“DeepEE”按钮
